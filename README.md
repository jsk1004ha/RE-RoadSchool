# RE路School 데모사이트

**RE路School — 학업중단 청소년을 위한 AI 복귀·진로 경로 설계 플랫폼** 데모입니다. 3~5분 자기진단 응답을 바탕으로 학업·진로·상담·지원기관 연결 경로를 보여주는 프론트엔드 전용 시연 앱입니다.

## 실행

```bash
npm install
npm run dev
npm run test
npm run build
```

## 구현 범위

- React + TypeScript + Vite + Tailwind CSS
- Recharts 기반 레이더/막대/게이지/도넛 차트
- Lucide React Icons, Framer Motion 전환
- localStorage 기반 설문/결과 임시 저장
- 랜딩 → 설문 → 분석 중 → 결과 → 로드맵 → 지원기관 → 리포트 흐름
- 지역별 지원센터 필터와 대표 예시 데이터
- 설문 시작 전 지역 선택 및 심각도 상 결과 시 선택 지역 상담소 우선 연결

## 비임상 규칙 기반 분석

이 데모는 실제 AI API나 임상 진단 도구가 아니라, 발표용 규칙 기반 분석을 사용합니다.

- 6개 요인 × 3문항, 각 문항 0~3점
- `rawScore`: 요인별 0~9점
- `rawPercent`: `Math.round(rawScore / 9 * 100)`이며 심각도와 유형 분류에 사용
- `displayPercent`: 기본 데모 화면의 시각적 완성도를 위한 보정 표시값이며 진단 점수가 아닙니다.
- 결과는 최종 판단이 아니라 상담자와 계획을 세우기 위한 **상담 참고자료**입니다.

## 데모/각색 데이터

요청 프롬프트의 기본 사용자 `김리로드`, 지역 `전북 전주시`, 기본 표시 점수 65/55/82/48/25/60은 시연 흐름을 자연스럽게 보여주기 위한 데모/각색 데이터입니다. 설문 문항은 학교 밖 청소년 지원 서비스 범주와 공개 자료의 주요 이슈를 참고해 만든 데모 자기진단 문항이며, 표준화된 심리검사가 아닙니다.

## 지역별 지원센터 데이터

지원기관 데이터는 지역별 대표 예시입니다. 모든 지역 센터를 빠짐없이 수록한 공식 디렉터리가 아니며, 실제 이용 전 공식 누리집 또는 1388로 최신 정보를 확인해야 합니다. 앱의 지역 필터는 발표 시연을 위해 각 지역마다 최소 1개 이상의 대표 예시가 보이도록 구성했습니다. 설문 시작 시 선택한 지역은 심각도 상 요인이 나타났을 때 해당 지역 청소년상담복지센터 1388 연결을 먼저 보여주는 데 사용됩니다.
